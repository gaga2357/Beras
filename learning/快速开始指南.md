## 🚀 开始步骤

### 1. 阅读学习路径（30分钟）
```bash
# 仔细阅读这个文件，理解整体框架
cat learning/README.md
```

**重点关注：**
- 每个阶段的学习目标
- 关键问题思考
- 理论知识

### 2. 准备知识（1-2天）
```bash
# 查看需要掌握的知识点
cat learning/知识点清单.md
```

**建议：**
- 如果数学基础薄弱，先看推荐资源
- 可以边学边做，不用全部学完再开始

### 3. 开始第一个练习（2-3天）
```bash
# 打开第一个练习文件
vim learning/exercises/01_tensor.py
```

**工作流程：**
1. 阅读文件顶部的学习目标和关键概念
2. 仔细阅读每个函数的 TODO 注释
3. 思考"关键问题思考"中的问题
4. 实现标记为 `# TODO` 的部分
5. 运行测试验证实现

### 4. 测试你的实现
```bash
# 运行练习文件中的测试
python learning/exercises/01_tensor.py

# 或运行专门的测试文件
python learning/tests/test_01_tensor.py
```

### 5. 记录学习进度
```bash
# 更新进度跟踪表
vim learning/学习进度跟踪.md
```

**建议记录：**
- 学习时间
- 关键收获
- 遇到的问题和解决方案
- 还不理解的地方

### 每个阶段的学习流程
```
1. 阅读理论知识（30分钟）
   ↓
2. 思考关键问题（30分钟）
   ↓
3. 实现代码（2-3小时）
   ↓
4. 运行测试（30分钟）
   ↓
5. 梯度检查（30分钟）
   ↓
6. 对比参考答案（30分钟）
   ↓
7. 总结反思（30分钟）
```

### 遇到困难时
1. 📖 重读理论知识部分
2. 💭 思考关键问题
3. 🔍 使用梯度检查工具调试
4. 📝 记录问题
5. 🆘 查看 `solutions/` 中的参考答案
6. 💬 向导师请教

## ❓ 常见问题

### Q: 我数学不好，能学吗？
A: 可以！边学边补。重点掌握：
- 矩阵乘法
- 链式法则
- 基本导数

### Q: 需要多长时间完成？
A: 因人而异：
- 有基础：2-3 周
- 零基础：4-6 周
- 每天 2-3 小时

### Q: 必须按顺序学习吗？
A: 第一次学习建议按顺序。后续可以跳着复习。

### Q: 什么时候可以看参考答案？
A: 建议：
1. 先自己尝试实现
2. 遇到困难思考 30 分钟以上
3. 实在不行再看答案
4. 看完后自己重新实现一遍
